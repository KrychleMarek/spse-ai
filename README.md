# 2025MP42 SPŠE AI Chatbot pro školu


## Jak Spustit
1. přidat OpenAI API klíč do souboru config.json v home dir.

2. **Build image** (v home dir projektu)
    ``` 
    docker compose up --build -d
    ```
3. **Zobrazení stránky**

    a. Administrátorksý panel: ``` http://localhost:8080/ ```
   
    b. Uživatelské rozhraní:   ``` http://localhost:8000/ ```
